# 프로젝트: gif 아카이브
## 기능:
- gif 파일에 이름과 태그를 붙여 서버 DB에 저장(사용자는 이름과 비밀번호를 입력해야 한다)
- 자신이 저장한 gif 파일을 삭제하는 기능(업로드할때 입력했던 비번 입력해서 일치하면 삭제, 일치 안하면 오류)
- 저장할때 입력했던 태그, 이름을 바탕으로 검색창에 검색어를 입력해서 이에 해당되는 gif 조회하는 기능(한 페이지당 10개씩 보여주고 여러 개의 페이지 존재 가능)

### 필드
- [x] DB에 저장되는 고유 id
- [x] 사용자가 저장할때 입력한 gif의 이름 
- [x] 실제 서버에 저장될 파일명 (UUID 등)
- [x] 업로더 이름
- [x] 사용자가 입력한 비밀번호를 암호화한 해시값
- [x] 쉼표로 구분된 태그 문자열

- [x] 파일 크기(바이트 단위)
### 메서드
- [ ] 사용자가 문자열 형태로 입력한 값에 빈 문자열이 있는지 검증하는 메서드
-> 오류: 빈 문자열을 입력할 수 없습니다.
- [ ] 입력받은 gif 정보를 검증하는 메서드
- [ ] 입력받은 유저 정보를 검증하는 메서드
