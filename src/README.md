# 프로젝트: gif 아카이브
## 기능:
- gif 파일에 이름과 태그를 붙여 서버 DB에 저장(사용자는 이름과 비밀번호를 입력해야 한다)
- 자신이 저장한 gif 파일을 삭제하는 기능(업로드할때 입력했던 비번 입력해서 일치하면 삭제, 일치 안하면 오류)
- 저장할때 입력했던 태그, 이름을 바탕으로 검색창에 검색어를 입력해서 이에 해당되는 gif 조회하는 기능(한 페이지당 10개씩 보여주고 여러 개의 페이지 존재 가능)
